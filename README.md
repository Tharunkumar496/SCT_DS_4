# 🚦 US Accidents Data Analysis
## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on the US Accidents dataset to identify trends, patterns, and insights related to road accidents across the United States. The analysis focuses on accident frequency, time-based patterns, and various road and environmental factors.
## 🎯 Objectives
- Understand the structure of the dataset.
- Clean and preprocess the data.
- Analyze accident occurrences by time.
- Visualize accident patterns using charts and graphs.
- Extract meaningful insights from the data.
## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook
## 📂 Dataset
The dataset contains information about traffic accidents in the United States, including:
- Severity
- Start Time
- City
- State
- Weather Conditions
- Traffic Signals
- Junctions
- Crossings
- Railway Crossings
- Sunrise/Sunset Information
## 📊 Analysis Performed
### Data Cleaning
- Removed unnecessary columns.
- Handled missing values.
- Converted date and time columns into proper datetime format.
### Feature Engineering
- Extracted the hour from accident timestamps.
- Created time-based features for analysis.
### Data Visualization
- Histogram of accidents by hour.
- Distribution analysis of accident occurrences.
- Trend identification through graphical representations.
## 📈 Key Findings
- Most accidents occur during peak traffic hours.
- Accident frequency varies throughout the day.
- Traffic-related infrastructure can influence accident occurrence.
- Time-based analysis helps identify high-risk periods.
